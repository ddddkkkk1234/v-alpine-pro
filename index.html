컴퓨터 비전 기반 실시간 역학 분석

알파인 스키(Alpine Ski) 분석 시스템은 구글의 MediaPipe 프레임워크를 기반으로 구축되었습니다. 초당 30프레임 이상의 속도로 33개의 주요 신체 관절 포인트를 추적하며, 특히 스키 주행 시 가장 큰 부하가 걸리는 무릎 관절의 가동 범위를 실시간으로 계산합니다.

부상 예방을 위한 데이터 가이드

스포츠 의학 연구에 따르면, 급격한 턴이나 착지 시 무릎 각도가 120도 이하로 깊게 굽혀지면서 발생하는 회전력은 전방십자인대(ACL) 파열의 주요 원인이 됩니다. 본 시스템은 이러한 위험 구간을 시각화하여 사용자가 스스로 안전한 범위를 인지하도록 돕습니다.

개인정보처리방침 및 데이터 보안

본 서비스는 사용자의 프라이버시를 최우선으로 보호하기 위해 다음과 같은 보안 원칙을 준수합니다.

  • ✅ 영상 비저장 원칙: 카메라를 통해 입력되는 모든 영상 데이터는 실시간 분석 직후 즉시 폐기됩니다. 어떠한 형태의 녹화물도 서버로 전송되거나 저장되지 않습니다.
  • ✅ 기기 내 로컬 처리: 모든 AI 연산 및 데이터 처리는 사용자의 브라우저 환경 내에서만 수행됩니다. 외부 클라우드로 데이터가 유출되지 않는 안전한 구조입니다.
  • ✅ 카메라 접근 권한: 카메라 접근 권한은 오직 관절 분석 기능을 위해서만 사용되며, 사용자가 분석 버튼을 누를 때만 활성화됩니다.
